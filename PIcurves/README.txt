Thank you for downloading the stoch_locus by Eric Thrane and Joe Romano.
Contact: eric.thrane@ligo.org or joseph.romano@ligo.org.

This download contains Matlab code for making power-law integrated plots.  The
primary code is called PIcurves.m.  A detailed description of PIcurves, 
and the different options for running it, is provided in the header of 
PIcurves.m.  To see an example of how to run it, please see example_plot.m.  

This material is part of LIGO DCC P1300115.
